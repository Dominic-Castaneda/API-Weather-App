# Weather App (React Native + Expo)

Welcome to the group weather app project! This repo contains the React Native front-end built with Expo.

> 🚨 Important: All code **must be submitted via branches and reviewed by Dominic** before being merged into `main`.

---

## 🚀 Getting Started

### Step 1: Clone the Repo

We’re using **GitHub Desktop** for version control.

1. Open **GitHub Desktop**.
2. Click **"File" > "Clone Repository"**.
3. Find the repo and clone it to your local machine.

---

### Step 2: Open in VS Code

Once cloned:

1. Open **VS Code**.
2. Open a **new terminal** in VS Code (Terminal > New Terminal).

---

### Step 3: Navigate to the Client App

In the terminal, type:

```bash
cd Weather-Client
```

This folder contains the actual React Native code.

---

### Step 4: Start the App

Then run:

```bash
npx expo start
```

This will launch the Expo Dev Tools. You can scan the QR code with your phone (using Expo Go) or run it on an emulator.

---

## 🌱 Branch Workflow & Ruleset

### ✅ Do NOT work directly on the `main` branch.

We are using a **pull request review system**. Here's the process:

1. **Create a branch** for your feature or fix (in GitHub Desktop):
   - Example: `feature-search-screen`, `fix-forecast-bug`, etc.
   - From GitHub Desktop: `Branch > New Branch`

2. **Make your changes** in VS Code.

3. **Commit and push** your work from GitHub Desktop.

4. **Open a Pull Request** on GitHub.
